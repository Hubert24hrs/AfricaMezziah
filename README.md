# Africa Mezziah E-Commerce

## Overview
Africa Mezziah is a premium e-commerce platform for a female clothing brand, featuring 3D product previews, AI-powered ordering assistance, and a modern futuristic design.

## Structure
This project is a monorepo containing:
- `apps/frontend`: Next.js 14 application (Frontend)
- `apps/backend`: NestJS application (Backend API)

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm / yarn / pnpm
- PostgreSQL

### Installation
1.  Clone the repository.
2.  Install dependencies for both apps:
    ```bash
    cd apps/backend && npm install
    cd apps/frontend && npm install
    ```

### Running Locally
- **Backend**: `cd apps/backend && npm run start:dev`
- **Frontend**: `cd apps/frontend && npm run dev`

## Tech Stack
- **Frontend**: Next.js, TailwindCSS, React Three Fiber.
- **Backend**: NestJS, PostgreSQL.
- **AI**: OpenAI / Gemini integration.
